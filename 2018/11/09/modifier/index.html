java修饰符的限制范围

发表于 2018-11-09 | 分类于 java

以下为Java控制可见性的四个修饰符的总结

  1. 仅对本类可见 – private
  2. 对本包可见 – 默认（无修饰符）
  3. 对本包及所有子类可见 – protected
  4. 对所有类可见 – public
访问修饰符 本类 同包 子类 其它
