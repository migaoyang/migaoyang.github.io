java多态、向上转型、动态绑定

发表于 2018-11-14

多态：意味着允许不同类的对象对同一消息做出不同的响应。
在Java中，按照广泛的范围来说，多态 可以分为 编译时多态（也叫设计时多态，
一般通过方法的重载来实现） 与 运行时多态（直到ｊａｖａ程序运行时候，系统才能根据调用方法的实例的类型来去决定去具体调用哪个方法）。
我们在Java中所说的多态就是运行时多态，在程序运行时动态决定调用哪个方法。
两个必要条件：

  • 满足继承关系
  • 父类引用指向子类对象 
public class Animal{
  private String name;
  private int month;
  public Animal(){  
  }

  public  Animal(String name,int month){
    this.name = name;
    this.month = month;
  }

  //name与month相对应的get、set方法
  //......

  public void eat(){
    System.out.println("动物吃东西");
  }
}

public class Cat extends Animal{
  private double weight；
  public Cat(){
  }

  public Cat(String name,int month,double weight){
    super(name.month)
    //this.setMonth(month);
    this.weight = weight;
  }

  //weight的get与set方法
  //......

  public void run(){
    System.out.println("小猫在跑")
  }
  @Override
  public void eat(){
    System.out.println("小猫吃鱼")
  }
}

public class Test {
  public static void main (String[] args){
    Animal one = new Animal();
    Animal two = new Cat();  //向上转型、隐式转型、自动转型》》父类引用指向子类实例，小类型转向大类型。

}

Animal two = new Cat();
此处为向上转型、隐式转型、自动转型 ———>> 父类引用指向子类实例，小类型转向大类型。可以调用子类重写父类的方法以及父类派生的方法。但是子类特有的方法无法使用。

public class Test {
  public static void main (String[] args){
    Animal one = new Animal();
    Animal two = new Cat();  //向上转型、隐式转型、自动转型》》父类引用指向子类实例，小类型转向大类型。
    Animal three = new Dog();
    one.eat();
    two.eat();
    three.eat();
  }
}

Cat temp = (Cat)two;
向下转型、强制类型转换 ———>>子类引用指向父类对象，此处必须进行强转，可以调用子类特有的方法。
不能Dog temp = (Dog)two；

two在引用的时候，指向的是Dog的实例空间，所以Cat temp = (Cat)two; 相当于还原为原来的类型。
